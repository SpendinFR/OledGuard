# OledGuard 0.6.0

Cette version revient au rendu carré original, sans flou spatial.

## Comportement

- grille carrée nette, 48 px par défaut ;
- fondu temporel fluide vers le noir ;
- changement de contenu révélé immédiatement ;
- zones statiques noires après 30 secondes ;
- passage de souris maintenu visible 30 secondes ;
- toutes les cases d’un même trajet de souris s’éteignent ensemble ;
- petite fermeture des trous entre cases actives, sans agrandir un point isolé ;
- `Ctrl+Alt+O` active ou désactive ;
- `Ctrl+Alt+R` révèle tout pendant 10 secondes.

## Construction

Lancer `build.cmd`, ou pousser sur GitHub pour utiliser le workflow Actions.
